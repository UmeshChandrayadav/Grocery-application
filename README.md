# 🛒 Grocery App

A modern Grocery Application built using React.js that allows users to browse products, manage their cart, and perform product operations efficiently.

---

## 🚀 Features

* 🛍️ Browse grocery products (fruits, vegetables, dairy, etc.)
* 🛒 Add to Cart & Remove from Cart
* 🔐 User Authentication (Login & Signup)
* ➕ Add Product
* ✏️ Update Product
* ❌ Delete Product
* 📱 Responsive UI design

---

## 🛠️ Tech Stack

### Frontend:

* React.js
* Tailwind CSS
* JavaScript

### Backend:

* JSON Server (Mock API)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/grocery-app.git
```

### 2️⃣ Navigate to project

```
cd grocery-app
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Start frontend

```
npm run dev
```

### 5️⃣ Start backend (JSON Server)

```
npx json-server --watch Backend/db.json --port 3000
```
