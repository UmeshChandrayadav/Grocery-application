# Grocery App

A React-based Grocery Application that allows users to browse products, manage cart, and perform CRUD operations.

## 🚀 Features
- User Login & Signup
- Add / Update / Delete Products
- Shopping Cart functionality
- Responsive UI

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- JSON Server (Backend)

## 📁 Project Structure
- Components
- Pages
- Context API

## ▶️ Run Locally
npm install  
npm run dev
